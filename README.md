# propellerhead-andy


## Prerequisite
Download and install .net 5 on your machine
https://dotnet.microsoft.com/download/dotnet/5.0

## Starting server
Go to project directory and run `donet run`
View list of APIs on `https://localhost:5001/swagger/index.html`
